# Excel / CSV Data Cleaner

A simple Python tool that cleans Excel and CSV files by removing empty rows and duplicate rows.

## Features

- Supports CSV files
- Supports Excel files
- Removes empty rows
- Removes duplicate rows
- Saves a cleaned output file automatically

## Technologies Used

- Python
- Pandas
- OpenPyXL

## How to Run

1. Install the required libraries:

```bash
python -m pip install pandas openpyxl
